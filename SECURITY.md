# Security Policy

## Reporting a Vulnerability or Concern
If you believe you’ve found a security or privacy issue in this repository (including the CI workflows), please report it responsibly.

- Create a new GitHub Issue and add the `security` label, or
- Contact the maintainer privately via the email listed on their GitHub profile.

Please avoid disclosing sensitive details publicly until we’ve assessed and addressed the issue.

## Dependencies and CI
- Dependabot is configured to monitor GitHub Actions.
- OpenSSF Scorecard workflow provides automated security posture checks.
